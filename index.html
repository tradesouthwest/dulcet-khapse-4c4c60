Healthy Work Habits Extension

This Firefox extension promotes healthy work habits with customizable break reminders.

Installation Instructions

  1. Open Firefox and navigate to about:debugging
  2. Click "This Firefox" in the left sidebar
  3. Click "Load Temporary Add-on"
  4. Navigate to this folder and select manifest.json
  5. The extension will appear in your browser toolbar

Features

  • ⏰ Automatic 25-minute work intervals
  • 🎯 Customizable break reminders
  • 💪 Motivational health messages
  • 📊 Progress tracking and statistics
  • 🔧 Multiple notification styles
